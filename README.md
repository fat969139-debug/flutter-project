# flutter-project
## دليل لمشاهدة الأفلام بتصميم يشبه Netflix



##  فكرة المشروع
تطبيق دليل لمشاهدة الأفلام بواجهة مستخدم عصرية
مستوحاة من تصميم Netflix، يتيح للمستخدم
استعراض الأفلام ومعرفة تفاصيلها.

##  مميزات التطبيق
- عرض قائمة الأفلام
- البحث عن الأفلام
- صفحة تفاصيل لكل فيلم
- عرض اقتراحات لأفلام مشابهة داخل شاشة التفاصيل
- جلب البيانات من الإنترنت (API)
- عرض صور الأفلام من الشبكة
- تصميم حديث مستوحى من Netflix
- واجهة سهلة وسلسة
- استخدام Animations لواجهة أجمل


##  الأدوات والتقنيات المستخدمة
- Flutter
- Dart SDK ^3.7.2

## المكتبات المستخدمة
- cupertino_icons  
- cached_network_image (لعرض الصور من الإنترنت)
- http (لجلب البيانات من API)
- lottie (لإضافة الرسوم المتحركة)
- iconsax (أيقونات حديثة)
- intl (تنسيق الوقت والتاريخ)
